Generate response from models by leading providers. Add Knowledge Base ID for Retrieval Augmented Generation. {{ className: 'lead' }}

<Properties>
  <Property name="modelId" type="string" required="true">

## List all models {{ tag: 'GET', label: '/models' }}

<Row>
  <Col>

    This endpoint allows you to add a new contact to your contact list in AI Library. To add a contact, you must provide their AI Library username and phone number.

    ### Required attributes

    <Properties>
      <Property name="username" type="string">
        The username for the contact.
      </Property>
      <Property name="phone_number" type="string">
        The phone number for the contact.
      </Property>
    </Properties>

    ### Optional attributes

    <Properties>
      <Property name="avatar_url" type="string">
        The avatar image URL for the contact.
      </Property>
      <Property name="display_name" type="string">
        The contact display name in the contact list. By default, this is just the username.
      </Property>
    </Properties>

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/contacts">

    ```bash {{ title: 'cURL' }}
    curl https://api.AI Library.chat/v1/contacts \
      -H "Authorization: Bearer {token}" \
      -d username="FrankMcCallister" \
      -d phone_number="1-800-759-3000" \
      -d avatar_url="https://assets.AI Library.chat/avatars/frank.jpg"
    ```

    ```js
    import ApiClient from '@example/AI Library-api'

    const client = new ApiClient(token)

    await client.contacts.create({
      username: 'FrankMcCallister',
      phone_number: '1-800-759-3000',
      avatar_url: 'https://assets.AI Library.chat/avatars/frank.jpg',
    })
    ```

    ```python
    from AI Library_api import ApiClient

    client = ApiClient(token)

    client.contacts.create(
      username="FrankMcCallister",
      phone_number="1-800-759-3000",
      avatar_url="https://assets.AI Library.chat/avatars/frank.jpg",
    )
    ```

    ```php
    $client = new \AI Library\ApiClient($token);

    $client->contacts->create([
      'username' => 'FrankMcCallister',
      'phone_number' => '1-800-759-3000',
      'avatar_url' => 'https://assets.AI Library.chat/avatars/frank.jpg',
    ]);
    ```

    </CodeGroup>

    ```json {{ title: 'Response' }}
    {
      "id": "WAz8eIbvDR60rouK",
      "username": "FrankMcCallister",
      "phone_number": "1-800-759-3000",
      "avatar_url": "https://assets.AI Library.chat/avatars/frank.jpg",
      "display_name": null,
      "conversation_id": "xgQQXg3hrtjh7AvZ",
      "last_active_at": null,
      "created_at": 692233200
    }
    ```

  </Col>
</Row>

      -H "Authorization: Bearer {token}"
