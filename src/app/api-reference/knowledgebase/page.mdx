## Create a Knowledge Base {{ tag: 'POST', label: '/v1/knowledgebase' }}

<Row>
  <Col>

    ### Request body
      <Properties>

        </Property>

  </Col>

    )

  </Row>

## Add Source to Knowledge Base  {{ tag: 'PUT', label: '/v1/knowledgebase/{knowledgeId}' }}

<Row>
  <Col>

    Add a list of files to a knowledge base.

    ### Request body
      <Properties>
      <Property name="type" type="string" required="true">
          Options - `docs`
        </Property>
        <Property name="meta" type="json">
          Meta data of the knowledge base.
        </Property>
        <Property name="options" type="object">
          Meta data of the knowledge base.
        </Property>
        <Property name="options > urls" type="array">
          List of URLs to add to the knowledge base.

          Accepted file types are PDF, DOCX, PPTX, XLSX, CSV, TXT. 
          
          Processing PDF and TXT files are faster than other file types.
        </Property>
      </Properties> 

    ### Response
    Returns the status of the request.
    
    NOTE: The id returned in the response is NOT the id of the source itself. For information about this source, please use "List Sources".


  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="POST" label="/v1/agent/knowledgebase/{knowledgeId}">
    ```json {{'title': 'Body'}}
    {
        "type": "docs",
        "options": {
            "urls": [
                "https://assets.ctfassets.net/416ywc1laqmd/7oJfDqN7x2Bjj3bXiP3WHR/5866c87cd89c15987bbb7211bfc3f3cc/Update_11_17_23.pdf"
            ]
        },
        "meta": {
            "folder": "home"
        }
    }
    ```
    ```python
    urls = ["https://example.com/docs.pdf"]
    options = {"urls": urls}
    response = client.knowledge_base.add_source(
        knowledgeId="kb_abc123",
        type="docs",
        options=options,
        meta={"category": "documentation"}
    )
    ```
    </CodeGroup>
  
    ```json {{ title: 'Response' }}
    {
      "status": "processing", "id": 2036
    }
    ```

  </Col>
</Row>

## Delete Sources {{ tag: 'DELETE', label: '/v1/knowledgebase/{knowledgeId}/source' }}

<Row>
  <Col>
    Delete sources from a knowledge base.
    <Properties>
      <Property name="values" type="array" required="true">
          List of sources to delete.
      </Property>
      <Property name="delete_all" type="boolean">
          Delete all sources.
      </Property>
    </Properties>
  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="DELETE" label="/v1/knowledgebase/{knowledgeId}/source">
    ```json {{'title': 'Body'}}
    {
        "delete_all": false,
        "values": [
            "my first knowledge Source.pdf",
            //...
        ]
    }
    ```
    ```python
    source_names = [
      "my first knowledge Source.pdf",
      # ...
    ]
    response = knowledge_base.delete_sources(
          knowledgeId="kb_abc123",
          values=source_names
    )
    ```
    </CodeGroup>
    ```json {{ title: 'Response' }}
    {
        "message": "Deleted successfully"
    }
    ```
