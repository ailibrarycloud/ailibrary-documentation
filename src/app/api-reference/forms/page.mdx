# Forms

Forms are useful in extracting structured information from documents or even guide conversations to fill a form. {{ className: 'lead' }}

To use forms in AI Library, you need to 
1. Create a form with a schema. The schema defines the structure of the form, including the fields and their types. {{ className: 'lead' }}
2. Once you have created a form, you update your agent with `form_filling` and `form_id`. {{ className: 'lead' }}
3. To generate a structured response, use the `chat` endpoint with `response_format` set to `json`. {{ className: 'lead' }}
4. To extract information from a chat, use `chat` as is. Conversations will be guided to fill the form. {{ className: 'lead' }}

There are three useful types of fields in a form:
1. `string`: A text field.
3. `boolean`: A true/false field.
4. `string` with `enum`: A dropdown field.

`enum` and `boolean` values are useful in taking actions based on the user's response. {{ className: 'lead' }}

For example, if a boolean returns `true`, you can ask for more details. If an `enum` returns a specific value, you can take an action based (eg.  invoke another agent) on that value. {{ className: 'lead' }}

## Create Form {{ tag: 'POST', label: '/v1/form' }}

<Row>
  <Col>

    Create a new form.

    ### Request body

    <Properties>
      
      <Property name="title" type="string" required="true">
            Form title.
      </Property>
        <Property name="schema" type="object" required="true">
            Form schema. The schema defines the structure of the form, including the fields and their types. Here's a sample object
            ```
            {
                "field_name": { 
                    "type": "string", //can be string, number, boolean
                    "order": 1, 
                    "description": "What's your name?"  //This is the prompt for the agent
                    "enum": ["", "option1", "option2"] //Optional. If the field is an enum, you can provide the options here
                },
                /// ...
            }
            ```
        </Property>
    </Properties>



   

  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/v1/form">
     ```bash
        curl --location 'https:/api.ailibrary.ai/v1/form' \
        --header 'Content-Type: application/json' \
        --header 'X-Library-Key: [[X-Library-Key-masked-secret]]' \
        --data '{
            "title": "Test Form",
            "schema": {
                "name": { "type": "string" },
                "email": { "type": "email" },
                "years_of_experience": { "type": "number" },
                "years_experience_with_nextjs": { "type": "number" },
                "AI_experience": { "type": "boolean" }
            }
        }'
    ```
        </CodeGroup>
    <CodeGroup title="Response" tag="POST" label="/v1/form">

    ```json {{ title: 'Response' }}
    {
        "form_id": "form_12345", //form_id
        "title": "...",
        "schema": "...",
    }
    ```
 </CodeGroup>
  </Col>
</Row>
