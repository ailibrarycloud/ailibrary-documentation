# AI Library API Documentation

AI Library APIs can be used to add AI capabilities into any product. You can use the APIs to {{ className: 'lead' }}
- [generate](/generate) responses from any [AI model](/models) in multiple formats {{ className: 'lead' }}
- create [knowledge base](/knowledgebase) to ground LLMs to your content and generate relevant responses {{ className: 'lead' }}
- use [RAG utilities and connectors](/utilities/) to add data to knowledge base from a variety of sources to inject relevant information into your prompts. {{ className: 'lead' }}
